The Brief

Part 1: Codepen Account

The basis of this project will be to display code ‘snippets’ on a protfolio website. As a vehicle to host and deliver these snippets you are to create a ‘Codepen’ account at http://codepen.io/ and develop on this account all the bits and pieces of code created throughout the term.

  • Applicable Learning outcomes:
  • 72.1 Understand concepts of operating systems
  • 72.3 Be able to apply programming techniques

Part 2: Portfolio website

Turning theory into practice you are to assemble your ‘Codepens’ into a ‘Resource’ website. Use appropriate JavaScript and jQuery enhancements to showcase your coding proficiency and use a combination of js and advanced css to create an effective and engaging website. As well as pages for showcasing your ‘Codepens’, you will create an ‘about this site’ page/section explaining the project and how you built your site and an ‘author’ page/section to give information about yourself”.

This can be a single or multi-page site, whatever suits your intended experience you want to create. The goal of this project is to produce something that you can add to your CV as a representation of your JS coding skills.

  • Applicable Learning outcomes:
  • 72.1 Understand concepts of operating systems
  • 72.2 Be able to use software design techniques
  • 72.3 Be able to apply programming techniques
  • 72.4 Be able to implement formal testing procedures.

Part 3: Required JavaScript library

In addition to any JavaScript functionality you build into the site yourself you are REQUIRED to learn and include the following JavaScript library: Isotope ( http://isotope.metafizzy.co/ ) – Responsive Layout and Content Filtering

  • Applicable Learning outcomes:
  • 72.2 Be able to use software design techniques
  • 72.3 Be able to apply programming techniques
  • 72.4 Be able to implement formal testing procedures.

Part 4: Additional JavaScript functionality and effects.

You are to visit the jQuery plugin library Unheap ( http://www.unheap.com/ ) and incorporate at least two (2) additional effects on your site. Think of the overall experience and how you are engaging the user and choose effects or functionality that enhances this.

  • Applicable Learning outcomes:
  • 72.2 Be able to use software design techniques
  • 72.3 Be able to apply programming techniques
  • 72.4 Be able to implement formal testing procedures.
